# 🚀 OpenShift Disconnected Installation Course

Welcome to the **OpenShift Disconnected Installation** course repository.

This repository contains all the **steps, prerequisites, lab setup, and hands-on guidance** required to successfully install OpenShift in **restricted and disconnected environments**.

---

## 📌 About This Course

In many enterprise environments such as:
- Banks 🏦
- Government organizations 🏛️
- Secure production environments 🔐

Direct internet access is **not allowed**.

This course is designed to help you understand:
- How to plan for disconnected installations
- What prerequisites are required
- How to set up your lab environment
- Step-by-step OpenShift installation without internet access

---

## 🎯 What You Will Learn

- ✅ Understanding disconnected environments
- ✅ Key considerations before installation
- ✅ Required infrastructure and tools
- ✅ Image mirroring concepts
- ✅ Step-by-step OpenShift installation process
